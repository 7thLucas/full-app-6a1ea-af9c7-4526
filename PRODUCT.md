# EventFlow — Product Information

## Overview
EventFlow is an all-in-one dashboard for event planners to manage venues, vendors, guest lists, and timelines from a single unified interface.

## Target Users
- Professional event planners
- Wedding planners
- Corporate event coordinators

## Brand & Tone
- Brand name: EventFlow
- Tone: Professional, organized, and approachable. Clean and efficient language that instills confidence. Calm and structured — not flashy or loud.
- Voice: Helpful and clear. Prioritizes clarity and action over decoration.

## Core Features
1. **Unified Event Dashboard** — Central overview of all active events, upcoming milestones, and key stats.
2. **Venue Management** — Track venue details, availability, contracts, and contacts.
3. **Vendor Tracking** — Manage vendor profiles, agreements, payment status, and communication logs.
4. **Guest List Management** — Organize attendees, RSVPs, dietary needs, seating, and invitations.
5. **Timeline & Schedule Management** — Build and track event day schedules, milestones, and deadlines.

## Strategic Principles
- Clarity over complexity: Every screen should have a clear purpose.
- Data integrity: Accurate status tracking across all modules.
- Planner-first UX: Designed around real workflows of professional planners.
- Scalability: Supports single events to large portfolios of concurrent events.

## Anti-References
- Avoid overly playful or consumer-app aesthetics.
- Avoid cluttered or information-overloaded dashboards.
- Avoid generic SaaS blue — use a refined, event-industry-appropriate palette.

# EventFlow

## Problem
Event planners waste hours juggling venues, vendors, guest lists, and timelines across disconnected tools — WhatsApp threads, spreadsheets, and sticky notes. Things fall through the cracks, communications get lost, and events risk going sideways.

## Target Audience
Professional event planners, wedding planners, and corporate event coordinators who juggle multiple vendors, venues, and large guest lists simultaneously.

## MVP — First Feature
A unified event dashboard where planners can track venues, vendors, guest lists, and timelines for each event — everything in one place, nothing slipping through the cracks.

## App Name
EventFlow

## Goal
Deliver flawless events without the chaos — one dashboard, every moving piece under control.
